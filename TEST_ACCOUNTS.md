## 🏢 총괄 관리자 모드 테스트

### 👤 테스트 사용자: **시스템 관리자**
- **역할**: 전체 시스템 관리 및 모니터링
- **접속 방법**: 
  1. `admin@ahp-system.com`으로 로그인
  2. "총괄 관리자" 모드 선택
  3. 🏢 아이콘 클릭

### 📋 테스트 시나리오
1. **시스템 대시보드 확인**
   - [ ] 전체 사용자 수: 127명
   - [ ] 총 프로젝트 수: 23개
   - [ ] 활성 프로젝트: 8개
   - [ ] 시스템 성능 지표 확인

2. **사용자 관리 테스트**
   - [ ] 사용자 목록 조회
   - [ ] 새 사용자 추가 시뮬레이션
   - [ ] 사용자 권한 수정
   - [ ] 사용자 상태 관리

3. **프로젝트 모니터링**
   - [ ] AI 개발 활용 방안 프로젝트 상태 확인
   - [ ] 진행률 85% 표시 검증
   - [ ] 26명 평가자 참여 현황 확인

4. **시스템 설정 관리**
   - [ ] 자동 백업 설정 확인
   - [ ] 로그 시스템 접근
   - [ ] 데이터베이스 상태 모니터링

## 👤 개인 서비스 모드 테스트

### 👤 테스트 사용자: **프로젝트 관리자**
- **역할**: 개인 AHP 프로젝트 생성 및 관리
- **접속 방법**:
  1. `admin@ahp-system.com`으로 로그인
  2. "개인 서비스" 모드 선택
  3. 👤 아이콘 클릭

### 📋 테스트 시나리오
1. **개인 대시보드 확인**
   - [ ] 내 프로젝트 통계 (1개 활성)
   - [ ] 총 평가자 26명 표시
   - [ ] 평균 진행률 85% 확인

2. **프로젝트 관리 워크플로우**
   - [ ] 기존 "AI 개발 활용 방안" 프로젝트 접근
   - [ ] 새 프로젝트 생성 기능
   - [ ] 프로젝트 설정 단계별 진행

3. **단계별 설정 테스트**
   ```
   프로젝트 → 기준설정 → 대안정의 → 평가자배정 → 모델확정
   ```
   - [ ] 각 단계 진입 및 데이터 표시
   - [ ] 진행률 바 업데이트
   - [ ] 이전/다음 버튼 기능

4. **AI 개발 프로젝트 상세 확인**
   - [ ] 3개 상위 기준 (개발 효율성, 기술 혁신성, 실무 적용성)
   - [ ] 9개 세부 기준 표시
   - [ ] 9개 대안 (ChatGPT, GitHub Copilot 등) 확인
   - [ ] 26명 평가자 한국어 이름 표시

## 📊 평가자 모드 테스트 (시뮬레이션)

### 👤 테스트 사용자: **평가 참여자**
- **역할**: AHP 쌍대비교 평가 수행
- **접속 방법**: 
  1. 동일 계정으로 로그인 후 평가자 시뮬레이션
  2. 또는 브라우저 시크릿 모드로 별도 접속

### 📋 테스트 시나리오
1. **프로젝트 선택**
   - [ ] AI 개발 활용 방안 프로젝트 확인
   - [ ] 평가 방법 선택 (쌍대비교/직접입력)

2. **쌍대비교 평가**
   - [ ] 기준 간 쌍대비교 (3개 × 3개)
   - [ ] 대안 간 쌍대비교 (9개 × 9개)
   - [ ] 일관성 비율 계산 및 표시

3. **평가 결과 확인**
   - [ ] 개별 가중치 계산
   - [ ] 최종 순위 도출
   - [ ] 일관성 검증 통과

## 🚨 알려진 제한사항

### ⚠️ 데모 모드 제한
- **데이터 영속성**: 브라우저 새로고침 시 변경사항 초기화
- **실제 이메일**: 알림 발송 불가 (시뮬레이션만)
- **파일 업로드**: 제한적 기능

### ⚠️ Render 프리 요금제 제한
- **콜드 스타트**: 15분 비활성 후 서버 대기 상태
- **응답 지연**: 첫 API 호출 시 30초 지연 가능
- **동시 접속**: 제한된 동시 사용자 수

## 🆘 문제 해결

### 🔧 일반적인 문제
1. **로그인 실패**
   - 정확한 이메일/비밀번호 재확인
   - 브라우저 캐시 삭제
   - 시크릿/프라이빗 모드 사용

2. **데이터 로딩 지연**
   - Render 콜드 스타트 대기 (최대 30초)
   - 네트워크 연결 상태 확인
   - 브라우저 새로고침

3. **기능 오류**
   - 브라우저 콘솔 오류 메시지 확인
   - 다른 브라우저에서 재시도
   - GitHub Issues에 버그 리포트
