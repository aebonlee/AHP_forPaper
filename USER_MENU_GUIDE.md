## ⚖️ 평가자 메뉴 (Evaluator)

### 🏠 **평가자 홈**
- **나의 평가 현황**
  - 할당된 프로젝트 수
  - 완료한 평가 수
  - 진행 중인 평가
- **최근 활동**
  - 최근 완료한 평가
  - 대기 중인 평가
  - 일관성 검증 결과

### 📋 **할당된 프로젝트**
- **프로젝트 목록**
  - AI 개발 활용 방안 프로젝트
  - 프로젝트 설명 및 목적
  - 평가 마감일
- **프로젝트 상세 정보**
  - 평가 기준 미리보기
  - 대안 목록 확인
  - 평가 방법 안내

### ⚖️ **쌍대비교 평가**
- **단계별 평가 수행**
  - 1단계: 기준 간 쌍대비교 (3×3)
  - 2단계: 대안 간 쌍대비교 (9×9)
  - 실시간 일관성 검증
- **평가 지원 도구**
  - 판단 스케일 가이드
  - 이전 평가 참조
  - 자동 저장 기능

### 📝 **직접입력 평가**
- **가중치 직접 입력**
  - 백분율 방식 입력
  - 점수 방식 입력
  - 순위 방식 입력
- **입력 유효성 검증**
  - 합계 100% 검증
  - 논리적 일관성 확인
  - 자동 정규화

### 📊 **내 평가 현황**
- **평가 진행률**
  - 완료한 쌍대비교 수
  - 남은 평가 작업
  - 예상 소요 시간
- **평가 품질 지표**
  - 일관성 비율 (CR < 0.1)
  - 평가 소요 시간
  - 수정 횟수

### 📜 **평가 이력**
- **과거 평가 기록**
  - 참여한 프로젝트 목록
  - 평가 완료일
  - 개인 결과 요약
- **평가 통계**
  - 총 참여 프로젝트 수
  - 평균 일관성 비율
  - 평가 경험 레벨

### ✅ **일관성 검증**
- **실시간 CR 계산**
  - 기준별 일관성 비율
  - 허용 기준 (CR < 0.1)
  - 개선 제안
- **일관성 개선 도구**
  - 문제 판단 식별
  - 재평가 권장사항
  - 단계별 수정 가이드

### 📖 **평가 가이드**
- **AHP 방법론 소개**
  - 쌍대비교 원리
  - 판단 스케일 설명
  - 일관성의 중요성
- **평가 팁과 전략**
  - 효과적인 판단 방법
  - 일관성 유지 전략
  - 자주 묻는 질문

### ⚙️ **평가자 설정**
- **개인 환경 설정**
  - 언어 설정
  - 화면 테마
  - 알림 설정
- **평가 환경 설정**
  - 자동 저장 간격
  - 판단 스케일 선호도
  - 키보드 단축키

## 🚀 빠른 시작 가이드

1. `admin@ahp-system.com`으로 로그인
2. "총괄 관리자" 모드 선택
3. "📊 시스템 대시보드"에서 전체 현황 확인

3. "🏠 내 대시보드"에서 현황 확인
4. "📂 내 프로젝트"에서 AI 개발 프로젝트 접근

2. "📋 할당된 프로젝트"에서 AI 개발 프로젝트 선택
3. "⚖️ 쌍대비교 평가"에서 평가 수행
